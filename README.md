Registration-form
Using Express, ejs, Node js, MongoDB

To run download Source code here

https://github.com/Santhosh1802/Registration-form/archive/refs/heads/main.zip

To run use below command in terminal

    npm i

To run:

    node index.js

To view output:
Open your browser with

    http://localhost:3000
